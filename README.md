Baderaldeen Oil Services (BOS) - Corporate Web Platform

Overview
This repository contains the source code for the official enterprise web platform of Baderaldeen Oil Services (BOS). The platform is engineered to reflect the company's commitment to excellence, providing a robust, highly performant, and secure digital presence for our integrated oil and gas services in Libya.

Development & Ownership
This digital infrastructure was exclusively designed, developed, and is maintained by the BOS Internal Technical Team. 
All source code, architectural patterns, and proprietary assets contained within this repository are the exclusive intellectual property of Baderaldeen Oil Services.

Technical Architecture
The system is built upon a modern, enterprise-grade technology stack ensuring high scalability and security:
- Core Framework: Next.js (React) - App Router
- Type Safety: TypeScript
- Styling & UI: Tailwind CSS, Radix UI Primitives
- Internationalization: Dual-language support (Arabic RTL / English LTR)
- Animation & Interactions: Framer Motion

Getting Started (Authorized Personnel Only)

Prerequisites
- Node.js (v18 or higher)
- npm or bun package manager
